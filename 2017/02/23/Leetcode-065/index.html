065 Valid Number

Posted on 2017-02-23 | In Leetcode |

Leetcode 065，判断一个字符串是否是有效数字。

题目

Validate if a given string is numeric.

Some examples:
“0” => true

“ 0.1 “ => true

“abc” => false

“1 a” => false

“2e10” => true

Note: It is intended for the problem statement to be ambiguous. You should gather all requirements up front before implementing one.

解题思路

要判断字符串是否为有效数字，需要进行以下五种判断：

  1. 对于是数字的char，肯定是有效数字的组成部分之一，不可或缺。
  2. 对于小数点’.’，一个有效数字钟只能有一个小数点，可以是’3.’或是’.3’这样的形式，但是前后至少有一个地方有数字。且在’e’后面不可加小数点。
  3. 对于科学计数符号’e’，前面必须是数字，后面可以是正负号加数字，且后面不能跟小数
  4. 对于空格符号而言，空格加载数字的前后都没有问题，但是不能加在数字的中间
  5. 对于正负号’+’,’-‘可以加在e后，前面不能加数字，后面可以加数字或是小数点

有了上述判断，对于不在上述5中判断范畴内的一律认为是无效数字。遍历整个字符串得出判断。
